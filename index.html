Google Developer Student Clubs


The Egyptian E-Learning University

Google Developer Student Clubs is a Google Developers program for university students to learn mobile and web development skills , design thinking skills and leadership skills.

Technologies

Web Development

Learn the core foundations of a delightful web experience both for the user and developer. Stay up to tabs with emerging and trending technologies. Get access to a guided, tutorial and hands-on coding experience.

AI

Learn how to drive user engagement and retention with intelligent apps that are able to effectively serve users what they need without the fuss by providing these systems with the ability to automatically learn and improve from experience without being explicitly programmed.

Programming

Learn the basics of programming from A to Z for beginners, and learning resources for programming languages that you can rely on on your journey from zero to professionalism.

Network Engineer

Learn how to design, implement, and maintain cloud networking solutions. We will learn the process of designing, implementing, and managing cloud infrastructure, hybrid network communications, load balancing traffic, network routing, private access to services, network security, and monitoring.
